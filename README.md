# ♻️ Greener - Smart Recycling Mobile Application

## 📖 Overview

Greener is a smart recycling mobile application developed as a graduation project and awarded **1st Place**. The application aims to promote environmental awareness and encourage sustainable recycling practices through a user-friendly mobile experience.

Built using **Flutter** and **Firebase**, the application provides secure user authentication, real-time data management, and interactive features that help users track and improve their recycling activities.

## 🚀 Installation

1. Clone the repository

```bash
git clone https://github.com/RamaNael/greener-smart-recycling-app.git
```

2. Navigate to the project folder

```bash
cd greener-smart-recycling-app
```

3. Install dependencies

```bash
flutter pub get
```

4. Run the application

```bash
flutter run
```
